📊 Contador de Palavras

Um script Python que lê arquivos de texto, conta palavras e salva o resultado automaticamente

📂 Ver no GitHub ⬇️ Download
🐍 Python 📄 File I/O 📝 String Processing

📖Sobre o Projeto

Este é um projeto simples em Python com o objetivo de demonstrar habilidades fundamentais de programação. O script lê o conteúdo de um arquivo de texto, conta o número de palavras e salva o resultado em um novo arquivo.

🎯 Objetivo Educacional: Praticar conceitos básicos de manipulação de arquivos e lógica de programação.

✨ Funcionalidades

📖 Leitura de Arquivo

Lê o conteúdo completo do arquivo entrada.txt

🔢 Contagem Precisa

Processa o texto e conta o número exato de palavras

💾 Salvamento Automático

Salva o resultado em um arquivo saida.txt

✅ Feedback Visual

Exibe mensagem de sucesso no terminal

💻Como Usar

1. Clone o Repositório

git clone git@github.com:nataliaberbetviana/contagem-de-palavras-python.git

2. Navegue até a Pasta

cd contagem-de-palavras-python

3. Crie o Arquivo de Entrada

Crie um arquivo chamado entrada.txt na pasta do projeto e adicione o texto que você deseja contar.

Este é um pequeno exemplo de texto para verificar a funcionalidade do contador de palavras.

4. Execute o Script

python contador-palavras.py

5. Verifique o Resultado

Contagem de palavras concluída com sucesso se a contagem for de 21 palavras!
📄 Arquivo de Saída (saida.txt):
O número de palavras é: 21!
