🔐 Gerador de Senhas Simples

Um projeto Python que demonstra boas práticas de programação com geração aleatória de senhas seguras

📂 Ver no GitHub ⬇️ Download
🐍 Python 🧪 Pytest 🔀 Random 📝 String

📖Sobre o Projeto

Este é um mini-projeto em Python que demonstra o uso de funções, loops, condicionais e manipulação de strings. O programa solicita ao usuário o tamanho desejado para a senha e, em seguida, gera uma senha aleatória usando letras, números e caracteres especiais.

🎯 Objetivo Educacional: Demonstrar conceitos fundamentais de programação Python através de um projeto prático e útil.

✨ Características Principais

🔒 Senhas Seguras

Gera senhas com letras maiúsculas, minúsculas, números e caracteres especiais

🎲 Aleatoriedade

Cada senha gerada é única e imprevisível

✅ Validação

Tratamento de erros para entradas inválidas

🧪 Testes

Suite completa de testes com Pytest

💻Como Usar

1. Clone o Repositório

git clone git@github.com:nataliaberbetviana/gerador-de-senhas-simples.git

2. Navegue até a Pasta

cd gerador-de-senhas-simples

3. Execute o Script

python gerador.py

4. Siga as Instruções

Digite o tamanho da senha desejada: 12 O número de caracteres é: 12 A senha gerada é: aB3$xY9#mK2!

🧪Executando os Testes

O projeto inclui uma suite completa de testes usando Pytest:

Instalar Pytest

pip install pytest

Executar Testes

pytest
📋 Plano de Testes: O projeto inclui documentação detalhada dos testes realizados. Ver Plano de Testes →

def gerar_senha(quantidade_de_caracteres): # Conjunto de caracteres disponíveis caracteres_disponiveis = string.ascii_letters + string.digits + string.punctuation senha_final = "" # Loop para construir a senha for _ in range(quantidade_de_caracteres): caractere_sorteado = random.choice(caracteres_disponiveis) senha_final += caractere_sorteado return senha_final
